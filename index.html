Google Sheets API Sync Setup

To enable direct syncing to your Google Sheet, follow these simple steps:

  1. Open your Google Sheet: Google Sheet Link.
  2. In the menu, click Extensions > Apps Script.
  3. Delete any existing code inside the editor and paste the code below:
  1. Click Save (floppy disk icon), then click Deploy > New Deployment (top right).
  2. Select Web app as the deployment type (click the gear icon).
  3. Under Execute as, select Me. Under Who has access, select Anyone.
  4. Click Deploy, authorize spreadsheet permissions, and copy the generated Web app URL.
  5. Paste the copied URL in the sidebar input box on the left, and click Generate!
